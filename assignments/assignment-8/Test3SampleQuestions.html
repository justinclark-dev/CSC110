CSC 110 Test #3 Sample Questions

Coverage:

The coverage for Test #3 focuses on everything we have done since Test #2, especially working with files and manipulating lists. Expect to see nested loops, which did not appear on Test #2. Also expect to be asked to write code that involves problem-solving with loops. Review the assigned CodingBat problems to prepare for the "write some code" section of the test.

Types of Questions:

You may see any of the following 5 different types of questions on the test:

  1. "Evaluate the Expression" -- determine the value and data type of each expression shown.
  2. "Show the Output Produced" -- given a block of Python code, show what will be displayed the code is executed.
  3. "Write Some Code" -- given a description of the problem to be solved or the actions to be performed, write some Python code to accomplish the tasks.
  4. "Trace the Variables" -- given a program listing including decision, repetition structures and/or function definitions, and with lines numbered, trace the operation of the program and show the actual sequence in which lines are executed and the values of the variables as they change.
  5. "Miscellaneous Questions" -- multiple choice, true and false, and/or short answer questions focused on the chapter objectives (similar to quiz questions).

"Write Some Code": Write a Python program to produce the requested results.

  1. Write a program that reads a series of numbers from a file named 'profits.txt' and prints the highest, lowest and average value. The numbers are not necessarily integers. The program should not crash if the file is empty -- just show an appropriate message.
     
  2. Write a program that writes out to a file the squares of the numbers from 10 to 20 (100, 121, 144, ..., 400), one number per line.
     
  3. Write a function named 'reverse()' that takes one string parameter and returns a new string that has all the characters in the original string in reverse order. reverse('one two') --> 'owt eno' -- reverse('a') --> 'a' -- reverse('') --> '' You must us a loop to solve this problem; you may not use a 'slice' or the string object's 'reverse()' method.
     
  4. Write a function named 'merge()' that takes two string values as parameters and returns a new string value that "interleaves" the characters in the original strings. Here are some examples of the output produced:
    • merge('one', 'two') --> 'otnweo'
    • merge('one', 'fourteen') --> 'ofnoeurteen'
    • merge('fourteen', 'one') --> 'foonuerteen'
    • merge('one', '') --> 'one' # note: second parameter is an empty string
    • merge('', 'one') --> 'one' # note: first parameter is an empty string
