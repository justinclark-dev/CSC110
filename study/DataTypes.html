Data Types

The basic structure used for numeric calculations in any programming language is the expression.

Here is an example with numbers:

x = 14
y = x + 3

What value do you think the variable y has after these statements? Hopefully it seems obvious that y will have the value 17. The construction 'x + 3' is an example of an expression. In this case, it is made up of 3 things: a variable named x, a numeric literal: 3, and an operator: +. When the + operator has numbers for operands (instead of strings), it becomes the addition operator. All of this illustrates an important point:

Numeric data is not the same as string data.

So, the following two statements are not the same:

x = '14'  # could also be written like this:  x = "14"  --  either single or double quotes may be used
x = 14

The first stores in the variable x a 2-character string. The second stores the number 14 as an integer value.

The rest of this note describes 3 important rules for working with data.

Rule 1. It is important to always be aware of the type of data stored in a variable. Any type of data may be stored in any variable in Python (unlike some other programming languages). If the wrong data type is used, expressions may not produce the correct results. For example, consider the following:

str1 = '14'
num1 = 14
ans1 = str1 + str1
ans2 = num1 + num1

The value of ans1 is the 4-character string "1414". The value of ans2 is the number 28. Make sure you understand this!

What do you think will happen if you combine string and numeric data in the same expression? For example:

ans3 = str1 + num1

The answer in Python 3 is that a TypeError will be reported.  This kind of "mixed type" expression is not allowed in Python (although it is allowed in some other programming languages).

Rule 2. Use only numeric data in numeric expressions, and use only string data in string expressions.  Following this rule may require you to add code to convert between data types.  The built-in 'int()' function can be used to attempt to convert a string value to an integer.  The 'float()' function can be used to attempt to convert a string value to a floating-point number.  To convert a number to a string, 'str()' can be used for a basic conversion with default formatting, or 'format()' can be used to retain more control over how the number will be represented in the string (e.g. how many digits will be shown after the decimal point).  The 'int()', 'float()' and 'format()' functions are described in Chapter 2 in the Gaddis text.

Rule 3. Always explicitly convert user input to a number in order to obtain numeric data. Here is an example:

text = input('Give me a number: ')
num = float(text)

The first statement gets a string value from the user and stores it in the 'text' variable. The next statement converts the string value to a number by passing it to the 'float()' function; the function returns a float value that is stored in the 'num' variable. Another way of writing this is

num = float(input('Give me a number: '))

This compound statement uses a function call (input) as the argument for another function call (float). This means that the value returned by the input function is used as the argument for float. Compound constructions like this are common.
