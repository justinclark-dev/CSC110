Practice Activity -- Python Vocabulary -- SOLUTION

Write a short answer for each of the following:
  1. What is a variable? A variable is a named memory location where information can be stored by a computer program.
     
  2. Name three different data types. What kind of information does each represent? Type str (string) represents a sequence of characters (text). Type int (integer) represents an integer value (a positive or negative whole number or zero). Type float represents a floating-point number (a number that can have a fractional part).
     
  3. What Python tool can you use to get input from the user? the built-in function 'input()'
     
  4. Which two built-in functions allow you to convert a string value to a number? 'int()' and 'float()'
     
  5. What Python tool can you use to show output to the user? the 'print' function
     
  6. Explain how to write a Python comment. Begin the comment with the '#' character (outside of a literal string), and everything from that point through the end of the line will be treated as a comment. For example (comments are underlined):

    # This entire line is treated as a comment -- ignored by the Python interpreter.
    print('test') # Everything following the '#' character is a comment.
    print('# This line does NOT contain a comment. Why?')
     
  7. Show an operator precedence table for the 7 mathematical operators. The following table shows three levels of precedence. Operators on the same row have equal precedence. When an expression contains more than one operator with the same precedence, execution is according to operator associativity (left to right for arithmetic operators).

    **      (highest precedence)
    *   /   //   %
    +   -   (lowest precedence)
     
  8. Explain what an escape character is and give at least two examples. An escape character is a code, beginning with a backslash (\), that represents a special character inside a literal string. The following 'print' statement contains a literal string that includes two escape characters, \' and \n.

    print('This output isn\'t limited \nto one line.')
     
  9. Show two different ways to write a string literal.

    'surround the characters with single-quotes'
    "surround the characters with double-quotes"
     
  10. Show an example of how a Python statement can be extended over more than one line.

    print('This statement prints a long literal string value ' \
            + 'and the statement is split over two lines.')
     
  11. Explain the naming rules for variables.

    -> The name of a variable may not be a Python keyword.
    -> The name of a variable may not contain spaces.
    -> The first character must be a letter or an underscore (_) character. To follow convention, we will begin every variable name with a lower-case letter.
    -> Only lower-case letters (a-z), upper-case letters (A-Z), numerical digits (0-9) and the underscore character (_) may be used in a variable name. All identifiers are case-sensitive ('a' is not the same as 'A').
    -> To follow modern programming convention, we will use meaningful names for all variables.
    -> To follow Python or Java naming conventions, we will use one of the following formats for variable names:

    total_tax_credits
    totalTaxCredits ("camelCase")
     
  12. Read Section 2.1 in the text and identify two different tools that programmers use to design programs on paper before beginning to write actual code. Pseudocode and flow charts.
